# Anti-patterns (AI-slop)

Do **not** default to:

1. Purple-on-white or purple-to-indigo gradients
2. Warm cream + high-contrast serif + terracotta accent
3. Broadsheet newspaper layouts

Also avoid: unnecessary dark marketing mode, purple bias, glow effects, rounded-full pills, multi-layer shadows, emojis.

## Clutter signals

Pill clusters, stat strips, icon rows, boxed promos, schedule snippets, competing text blocks in hero.
