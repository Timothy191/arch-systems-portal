# Handoff routing

| Step type                 | Owner                                      |
| ------------------------- | ------------------------------------------ |
| Branded/landing visual    | `frontend-design` → `frontend-implementer` |
| Portal CRUD/UI            | `frontend-implementer`                     |
| Policy/docs/skills/agents | `ai-docs-sync`                             |
| Before done               | `sceptic`                                  |
| While blocked             | `idle-runner`                              |
| Formal score              | `agent-alignment-score` skill              |
| Quality gate              | `quality` / `verify` skills                |
| Multi-file spec           | `specs` skill first                        |

Every handoff names a real `.cursor/agents/` specialist when one exists.
