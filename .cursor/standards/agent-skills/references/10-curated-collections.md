# 5 Curated Skill Collections

Principle: **few but excellent** (少而精) — notable quality/origin, not exhaustive index.

## Categories

| Category               | Wiki | Audience                                |
| ---------------------- | ---- | --------------------------------------- |
| Skill Creation         | 5.2  | Skill authors                           |
| Official Organization  | 5.1  | Platform/service users                  |
| Document Processing    | 5.3  | docx/pptx/xlsx/pdf output               |
| Content Creation       | 5.4  | Writers, social creators                |
| Programming Assistance | 5.5  | Developers                              |
| Product Usage          | —    | Third-party product control             |
| Other Types            | —    | Domain specialists (marketing, science) |

## Origin types

- **Official org repos** — vendor-maintained (`anthropics/skills`, `vercel-labs/agent-skills`)
- **Community repos** — individual authors (`obra/superpowers`, `JimLiu/baoyu-skills`)

All categories use the **same skill folder structure** — domain differs, not format.

## Install any curated skill

```bash
npx skills add <owner/repo>
```

Then validate layout if contributing to this monorepo.
