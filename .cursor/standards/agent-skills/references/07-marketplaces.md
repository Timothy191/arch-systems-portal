# 3.1 Skill Marketplaces

| Platform                                         | Operator  | Differentiator                               |
| ------------------------------------------------ | --------- | -------------------------------------------- |
| [skills.sh](https://skills.sh/)                  | Vercel    | Usage leaderboard; pairs with `npx skills`   |
| [skillsmp.com](https://skillsmp.com/zh)          | Community | Auto-indexes GitHub; category/star/date sort |
| [context7.com](https://context7.com/?tab=skills) | Context7  | Integrated with context7 CLI                 |
| [skillstore.io](https://skillstore.io/zh-hans)   | Community | Security-audited; Chinese UI                 |
| [agentskills.me](https://agentskills.me/)        | Community | Cloud skill runtime                          |
| [skills.rest](https://skills.rest/)              | Community | Analysis + security reviews                  |

## Choosing

| Use case                        | Platform                     |
| ------------------------------- | ---------------------------- |
| Most adopted skills             | skills.sh                    |
| Browse all public GitHub skills | skillsmp.com                 |
| context7 CLI workflow           | context7.com                 |
| Security-reviewed listings      | skillstore.io or skills.rest |
| Run without local install       | agentskills.me               |
| Analyze specific skill          | skills.rest                  |
