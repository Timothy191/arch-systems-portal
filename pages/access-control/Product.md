# Product Specifications: Access Control Module

## 1. Overview & Business Value
Ensures site security and safety compliance by monitoring badge check-ins, tracking on-site personnel counts, and managing visitor logs in real-time.

## 2. Target Personas
- **Security Officers**: Need immediate alerts on unauthorized entry attempts and a real-time list of all personnel on-site.
- **HSE Inspectors**: Require instant access to headcount data in the event of an emergency evacuation.
- **Visitors**: Need a frictionless check-in and badging experience.

## 3. Core Features & Capabilities
- **Live Swipe Feed**: Continuous stream of gate transactions (Granted, Denied, Expired).
- **Visitor Registration**: Interface to register and pre-approve visitors, contractors, and vehicles.
- **Badge Management**: Portal to issue, renew, and revoke physical and digital RFID access badges.

## 4. Key Performance Indicators (KPIs) & Metrics
- **Current Headcount**: Real-time count of people currently inside the security perimeter.
- **Access Denials**: Number of unauthorized entry attempts.
- **Visitor Flow**: Turnaround time for contractor check-ins.

## 5. Security & Access Boundaries
Highly restricted. Only accessible by Security personnel and Administrators. RLS requires strict membership in the access-control department.
