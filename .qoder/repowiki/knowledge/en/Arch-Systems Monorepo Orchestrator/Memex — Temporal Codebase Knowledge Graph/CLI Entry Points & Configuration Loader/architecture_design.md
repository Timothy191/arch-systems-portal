Three-file leaf that forms the process boundary of the `memex` package:
- `__main__.py` is the minimal shim that imports and calls `memex.cli.main`, enabling `python -m memex`.
- `cli.py` is the single command dispatcher. It builds one `argparse.ArgumentParser` with a shared `--repo` parent parser and registers every subcommand (`init`, `watch`, `status`, `list`, `remove`, `pause`, `resume`, `serve`, `doctor`, `keys {add,list,revoke}`, `archive`, `stats`, `review`, `graph`, `cluster`, `memory-tool serve`) as either eager or lazy `ImportError`-guarded handlers. Each handler resolves an optional `repo_root` (defaulting to `.`), then delegates to async functions in sibling packages (`memex.watcher.daemon.run_daemon`, `memex.mcp_server.server.run_server`, `memex.graph.cluster_runner`, `memex.graph.archive`, `memex.cli_review`, `memex.cli_graph`, `memex.memory_tool.server`). Async commands are launched via `asyncio.run(...)` from the synchronous `main`. The `serve` subcommand additionally loads `.env` files (repo-local then `--env-file`) via `dotenv.load_dotenv(override=True)` and refreshes the module-level `_config` singleton before starting the server; `doctor` probes Python ≥3.11, `uv`, Docker, Neo4j connectivity, Gemini SDK presence, git hooks, daemon pid, stale edges, watcher health, and optional tracing summary.
- `config.py` defines the application configuration as Pydantic models (`Config`, `HarnessConfig`, `RetrievalConfig`) loaded from environment variables and an optional `<repo>/config.yaml` (resolved relative to `repo_root` when known). A module-level `_config` singleton is exposed through `get_config()`, and `canonical_repo_path()` normalises repo paths for cross-process key stability. An `MEMEX_INTROSPECTION_ONLY=1` mode lets the server start without live backends so tool enumerators can list capabilities.

Dependency direction: this leaf depends on `memex.watcher.*`, `memex.mcp_server.*`, `memex.graph.*`, `memex.memory_tool.*`, and `memex.config`; nothing inside those packages imports back into this leaf, keeping it a pure top-level facade.
