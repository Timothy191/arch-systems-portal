Requires `pnpm dev` / `pnpm build` / `pnpm start`; database migrations run via `npx prisma migrate deploy` (schema in `prisma/schema.prisma`). Environment variables include `GITHUB_ID`, `GITHUB_SECRET`, `NEXTAUTH_SECRET`, `DATABASE_URL`, `NEXT_PUBLIC_API_URL`, plus Neo4j/Three-related keys referenced by other layers. A helper script `scripts/verify-key.ts` exists for key validation.
