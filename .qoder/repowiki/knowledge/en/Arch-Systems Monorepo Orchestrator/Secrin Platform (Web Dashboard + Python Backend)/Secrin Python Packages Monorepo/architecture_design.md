Three sibling packages share one root via `sys.path` injection in `packages/__init__.py`, which prepends `tools/secrin/` so every child can import `packages.config.settings.Settings`. The `config` package is the sole source of truth for Neo4j, embedding provider, API, and feature-flag settings; both `cli` (Typer commands + agents) and `arc42gen` (orchestrator/generator) consume it through the same `Settings` model. There is no runtime cross-import between `cli` and `arc42gen` — they are independent entry points that happen to share the config contract. Package-level `__init__.py` files re-export public APIs (`cli` exposes Typer app, `arc42gen` exposes Orchestrator/Analyzer/Generator), while `__pycache__` is ignored by version control.
