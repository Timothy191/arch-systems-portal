Flat `components/` directory of self-contained Next.js client components (all prefixed with `"use client"`). Each file is a single-purpose presentational component: `HeroRotator` cycles through default + department panels; `DepartmentCard` renders a glassmorphic card with status badge, optional Sparkline, and action links; `ToolBanner` auto-rotates tool highlights via Framer Motion; `ProductionTrendWrapper` lazily loads the heavy chart via `next/dynamic` with `ssr: false` and a pulse placeholder; `Sparkline`, `AlertTicker`, `TrustLogos`, `HeroBackground` are supporting visuals. Data contracts come from `~/lib/departments` (`Department`, `TrendDataPoint`) rather than being fetched here. Styling is Tailwind + CSS variables from `@repo/ui` (`GlassCard`, `Badge`, `cn`), and icons come from `lucide-react`. There is no shared state or API layer — this feature is purely a composition surface consumed by a parent route.
