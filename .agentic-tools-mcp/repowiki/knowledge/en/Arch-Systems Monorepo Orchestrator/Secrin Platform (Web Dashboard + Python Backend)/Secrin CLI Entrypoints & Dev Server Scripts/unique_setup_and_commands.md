Run via Poetry console scripts declared in `pyproject.toml`: `poetry run server`, `poetry run arc42gen-api`, `poetry run arc42gen-worker`, `poetry run setup`, plus `migrate`, `ingest`, `arc42gen`, and `secrin`. The `setup` command requires a `.env.example` at the project root and will prompt for Neo4j credentials and embedding provider choice (Ollama / OpenAI / Sentence Transformer).
