Server mode requires a running Redis instance (URL from `packages.config.settings.Settings.REDIS_URL`); start the API with `uvicorn api:app --host 0.0.0.0 --port 8001` and run workers with `rq worker`. The CLI entry point is `arc42gen` (Click group) with subcommands `generate`, `analyze`, `diagrams generate`, `diataxis generate`, and `config init|validate`. Jobs can be submitted either directly via the API or through `arc42gen jobs submit --repo ... --owner ... --repo-name ... --source-owner ... --source-name ... [--wait]`.
