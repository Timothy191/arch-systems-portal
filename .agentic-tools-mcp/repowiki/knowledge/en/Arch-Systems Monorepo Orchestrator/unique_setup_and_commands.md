`pnpm quality` runs the full gate (Nx lint/type-check/test/lint:tokens/lint:css, root ESLint, Prettier, syncpack, knip); `pnpm policy:gen` regenerates `eslint.boundaries.cjs` from `tools/policy-definitions.ts`; `pnpm deploy {local|staging|production}` invokes `scripts/deploy.sh` which composes the appropriate docker-compose stack and triggers Nx builds; `pnpm monitor:grafana` starts Prometheus/Grafana via `docker-compose.monitoring.yml`.
