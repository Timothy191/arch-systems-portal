`make fetch-deps` downloads the ONNX model and ORT runtime before any build/test; `make build` produces `bin/sense` with version injected via `-ldflags`. Hermetic unit tests run via `make test-hermetic` (no network, no ONNX). Coverage gate requires ≥92% line+function coverage per file (`make cover`), enforced by `scripts/coveragegate`. Benchmark suite lives under `bench/` and is driven by `bench/global/bench.sh` plus driver scripts in `bench/drivers/`.
