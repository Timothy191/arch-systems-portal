Run as a CLI: `python -m memex.mcp_server --repo <path> [--transport stdio|http|both] [--host 0.0.0.0] [--port 8000]`. Set `MEMEX_INTROSPECTION_ONLY=1` to skip the Neo4j connectivity check (useful for directory sandboxes that only enumerate tools). When using the HTTP transport the running port is written to `<repo>/.memex/port`; set `MEMEX_MCP_TRANSPORT=sse` to opt into the deprecated SSE mode.
