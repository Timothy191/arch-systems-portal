# 🎨 Arch System — Design System Package

This package manages the design tokens, Tailwind presets, and design system governance for the Arch monorepo.

## 📦 Directory Structure

```bash
packages/theme/
├── src/
│   ├── css/
│   │   ├── variables.css      # Core CSS custom properties (source of truth)
│   │   └── variables-generated.css # Generated Style Dictionary variables
│   ├── tailwind/
│   │   └── preset.ts          # Tailwind preset extending theme configurations
│   ├── tokens/
│   │   ├── generated.ts       # Typed JS/TS token map (generated by scripts/generate-tokens.mjs)
│   │   └── motion.ts          # Framer Motion transitions and physics presets
│   └── index.ts               # Core JS entry point
├── scripts/
│   ├── generate-tokens.mjs    # Compiles CSS variables into generated.ts
│   └── validate-tokens.mjs    # Token syntax & usage checker
└── package.json
```

## 🏆 Enterprise-Grade Design Tokens

The Design System promotes semantic control and accessibility through a set of code-enforced tokens.

### 1. HUD & Overlays (Light-Frosted Scrims)

- `--color-bg-hud`: `oklch(0% 0 0 / 60%)` — Translucent overlay surface.
- `--color-border-hud`: `oklch(100% 0 0 / 10%)` — Hairline borders for overlays.
- `--color-text-hud-primary`: `oklch(100% 0 0)` — High-contrast white text.
- `--color-text-hud-secondary`: `oklch(100% 0 0 / 70%)` — Semi-translucent body text.
- `--color-text-hud-tertiary`: `oklch(100% 0 0 / 40%)` — Faint descriptive metadata.

### 2. Interactive Opacity Levels

- `--opacity-focus-dim`: `0.4` — For dimming surrounding non-active layers.
- `--opacity-disabled`: `0.38` — Enforces standard WCAG target contrast for disabled widgets.
- `--opacity-hover`: `0.08` — Subtle hover overlay tint.

### 3. Radial Corners & Radii

- `--radius-sm`: `4px` — Badges, small nested selectors.
- `--radius-md`: `6px` — Button components, input fields.
- `--radius-lg`: `8px` — Standard cards, dropdown containers, popovers.
- `--radius-xl`: `12px` — App docks, main modal windows.
- `--radius-full`: `9999px` — Rounded pills, status lights, user avatars.

## 🎴 GlassCard Unified Component (`@repo/ui/GlassCard`)

`GlassCard` consolidates standard glass, macOS-like window mockups, mouse-tracked spotlights, and conic rotating borders under a single unified component API.

### API Properties

```tsx
interface GlassCardProps extends HTMLMotionProps<"div"> {
  children: React.ReactNode;
  className?: string;
  hover?: boolean; // Enable standard scale/lift animation
  onClick?: () => void;
  accent?: "green" | "blue" | "red" | "none"; // Border highlight color on hover
  variant?: "default" | "window" | "spotlight" | "glowborder";
  title?: string; // Title text (only used in "window" variant)
  padding?: boolean; // Enable default padding spacing (24px)

  // Spotlight properties
  spotlightColor?: string; // Custom color for the radial hover gradient

  // GlowBorder properties
  animationDuration?: number; // Spin duration in seconds (default: 4s)
  colorPreset?: "nature" | "ocean" | "sunset" | "aurora" | "custom";
  gradientColors?: string[]; // Explicit array of hex/HSL values for custom gradient
  paused?: boolean; // Pause rotating animation
}
```

## 🚀 Migration Guide (Developers)

Legacy components `SpotlightCard` and `GlowBorderCard` have been consolidated into `GlassCard` and removed from disk. Update your imports and usages accordingly:

### 1. Migrating SpotlightCard

**Before:**

```tsx
import { SpotlightCard } from "@repo/ui/SpotlightCard";

<SpotlightCard spotlightColor="rgba(0, 122, 255, 0.1)">
  <div>Content</div>
</SpotlightCard>;
```

**After:**

```tsx
import { GlassCard } from "@repo/ui/GlassCard";

<GlassCard variant="spotlight" spotlightColor="rgba(0, 122, 255, 0.1)">
  <div>Content</div>
</GlassCard>;
```

### 2. Migrating GlowBorderCard

**Before:**

```tsx
import { GlowBorderCard } from "@repo/ui/GlowBorderCard";

<GlowBorderCard colorPreset="aurora" animationDuration={5}>
  <div>Content</div>
</GlowBorderCard>;
```

**After:**

```tsx
import { GlassCard } from "@repo/ui/GlassCard";

<GlassCard variant="glowborder" colorPreset="aurora" animationDuration={5}>
  <div>Content</div>
</GlassCard>;
```
